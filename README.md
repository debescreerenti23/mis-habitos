**Habit Tracker Pro** es una aplicación web ligera y funcional diseñada para ayudar a los usuarios a mantener la constancia en sus objetivos diarios. Este proyecto forma parte de mi portfolio personal como desarrollador, enfocándome en la manipulación del DOM, persistencia de datos y experiencia de usuario (UX).

▶️ **[VER DEMO EN VIVO](https://debescreerenti23.github.io/mis-habitos/)**

---

## ✨ Características Principales

* **Persistencia Local:** Los datos se guardan en el navegador mediante `localStorage`. Tus hábitos no se borran al cerrar la pestaña.
* **Calendario de Consistencia:** Visualización mensual con mapa de calor (estilo GitHub) para ver el progreso histórico.
* **Sistema de Rachas (Streaks):** Cálculo automático de días consecutivos completando todos los hábitos.
* **Modo Oscuro/Claro:** Interfaz adaptativa que respeta la preferencia del sistema o elección del usuario.
* **Feedback Visual:** Animación de confeti al completar el 100% de las tareas diarias.
* **Detalles Históricos:** Ventana modal interactiva para consultar qué hábitos se cumplieron en días pasados.

## 🛠️ Tecnologías Utilizadas

* **HTML5 & CSS3:** Diseño responsivo y variables CSS para el sistema de temas.
* **JavaScript (ES6+):** Lógica de negocio, gestión de fechas y manipulación dinámica del DOM.
* **Canvas Confetti:** Integración de librería externa para mejorar la gamificación.
* **Open Graph Meta Tags:** Optimizado para compartir en LinkedIn y redes sociales.
